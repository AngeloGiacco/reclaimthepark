# Contributing to Reclaim the Park

Thanks for your interest in helping reclaim London's parks for the public.

## How You Can Help

### Research
- File FOI requests using our [templates](comms/templates/)
- Add data to `research/data/` (CSV or JSON only)
- Write analysis notebooks in `research/analysis/`
- Document comparable projects in `research/comparables/`

### Design Competitions
- Help develop competition briefs in `competitions/briefs/`
- Build the submission platform in `competitions/platform/`
- Enter competitions when they launch

### Website
- Improve the campaign site in `website/`
- Keep it lightweight, accessible, and fast

### Outreach
- Draft press releases in `press/`
- Help with councillor and authority correspondence in `comms/`

## Guidelines

1. **Be factual** — all claims must be sourced and verifiable
2. **Be respectful** — we're working with authorities, not against them
3. **Keep it simple** — plain language, no jargon
4. **Use Markdown** — for all prose content
5. **Data as CSV/JSON** — no Excel or proprietary formats

## Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-idea`)
3. Commit your changes with clear messages
4. Open a pull request describing what you've done and why

## Code of Conduct

Be kind, be constructive, be honest. We're all here because we love London's parks.
