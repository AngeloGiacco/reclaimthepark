# Press & Media

## Structure

```
releases/       # Press releases (date-prefixed markdown)
coverage/       # Links to media coverage with summaries
media-kit/      # Downloadable assets for journalists
  fact-sheet.md         # One-page campaign fact sheet
  key-statistics.md     # Pull-out statistics for articles
  headshots/            # Campaign team photos (if public-facing)
  site-photos/          # Current car park photos
  renders/              # Competition renders (high-res, credited)
```

## Press Release Naming

```
YYYY-MM-DD_short-description.md
```

## Media Kit

The media kit should always be up to date and contain:

1. One-paragraph campaign summary
2. Five key statistics
3. 3–5 high-res renders (with credit/licensing info)
4. Site photos showing the current car park
5. Quote from campaign spokesperson
6. Contact details

## Copyright on Renders

Competition renders remain the intellectual property of their creators. When including in the media kit, always credit the team and confirm they have granted permission for press use. Include this in the competition brief terms.
