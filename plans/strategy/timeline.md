# Campaign Timeline

Master checklist for the Reclaim the Park campaign.

## Phase 1: Foundation (Current)

- [x] Set up repository and project structure
- [ ] Draft campaign messaging and key asks
- [ ] Identify priority parks and specific areas of concern
- [ ] Map stakeholders (see `stakeholder-map.md`)
- [ ] File initial FOI requests for land use, revenue, and footfall data
- [ ] Build campaign website

## Phase 2: Research & Evidence

- [ ] Analyse FOI responses and publish findings
- [ ] Document international comparables (King's Cross, Serpentine, etc.)
- [ ] Commission or crowdsource data visualisations
- [ ] Publish first research briefing

## Phase 3: Public Engagement

- [ ] Launch petition
- [ ] Run first design competition
- [ ] Host public event or walk
- [ ] Engage media — first press release and op-ed pitches

## Phase 4: Advocacy

- [ ] Present findings to Royal Parks Agency
- [ ] Meet with Westminster councillors
- [ ] Brief DCMS and Historic England
- [ ] Submit formal proposals for pilot programmes

## Phase 5: Implementation

- [ ] Support delivery of pilot activations
- [ ] Monitor and evaluate outcomes
- [ ] Publish lessons learned
- [ ] Plan next cycle
