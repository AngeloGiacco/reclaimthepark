## Statuses

- **Drafted** — ready to send
- **Sent** — submitted, awaiting acknowledgement
- **Acknowledged** — authority confirmed receipt
- **Extended** — authority has extended the deadline
- **Received** — response received (see `responses/`)
- **Appealed** — internal review requested
- **ICO Referral** — escalated to the Information Commissioner

## How to File

1. Use a template from `../templates/`
2. Send to the relevant authority's FOI address
3. Add a row to this tracker
4. Save the sent letter in `requests/`
5. When a response arrives, save it in `responses/`
