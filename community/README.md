# Community

Petition tracking, event coordination, and supporter engagement.

## Structure

```
petition/
  petition-text.md     # The petition wording
  platform.md          # Which platform (Change.org, 38 Degrees, custom)
  milestones.md        # Signature milestones and comms plan

events/
  YYYY-MM-DD_event-name.md  # Event plans and post-event notes

supporters/
  organisations.md     # Organisations that have endorsed the campaign
  individuals.md       # Notable public supporters (with their consent)
```

## Petition

The petition should target The Royal Parks Board of Trustees and call for:

1. Publication of the Euro Car Parks concession agreement
2. A public consultation on the site's future before any concession renewal
3. A commitment to exploring alternative uses that better serve the public
